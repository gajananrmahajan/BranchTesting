2. Explain the difference between findElement and findElements.
->findElement: Returns the first matching element on the current page.
findElements: Returns a list of all matching elements on the current page.

3. What is the purpose of the testing phase in SDLC?
->The testing phase ensures that the software product is free of defects and errors, meets the requirements, and is reliable and usable.

4. Explain method overloading and method overriding.
->Method Overloading: Multiple methods with the same name but different parameters within the same class.
Method Overriding: A subclass provides a specific implementation of a method that is already defined in its superclass.

5. What is an abstract class?
->An abstract class is a class that cannot be instantiated and may contain abstract methods (methods without a body) that must be implemented by subclasses.

6. Difference between PUT and POST
->PUT is like updating or replacing something, while POST is like adding something new.

7. If an element is getting displayed intermittently, how will write code such that no exceptions are to be thrown?
->To handle elements that are intermittently displayed and ensure no exceptions are thrown, you can use explicit waits in Selenium. Explicit waits allow you to wait for a specific condition to be met before proceeding with the next step in your code.
